# Blueprint Umbrella Website

Static B2B export website for Blueprint Umbrella.

## Local Preview

```bash
python3 -m http.server 8099
```

Then open:

```text
http://127.0.0.1:8099/index.html
```

## Main Pages

- `index.html`
- `products.html`
- `custom-umbrella.html`
- `solutions.html`
- `about.html`
- `blog.html`
- `download.html`
- `contact.html`
- `inquiry.html`

## Notes

- Static HTML/CSS/JavaScript project.
- Inquiry form includes frontend validation and a placeholder for backend integration.
- Multilingual frontend assets are stored in `locales/translations.js` and `i18n.js`.
